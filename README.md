智慧教育产品V1.0.1

### 安装
```
git clone https://github.com/Kelier/vue-element-ui.git
cd 项目名称
npm install
bower install
npm start
```

这里用gulp的browserify-sync开了一个本地服务器，自己打上路径应该没问题。 :bowtie:
